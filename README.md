## Description
This is the backend of a Social Media Api built with Mongo.db, Mongoose, and Express. You have the ability to create users, and with these users you can add/remove friends and create your own thoughts. You also have the ability to react to other user's thoughts. You can also update or remove any user from the database, as well as update/remove thoughts and reactions.
